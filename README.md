# 🧮 Number to Words Converter

A simple Blazor Web App and WinForm App that converts numbers (including decimals) into words.

For example:  
`42.66` → **Forty-Two point Six Six**

---

## 🚀 Features

- Convert whole and decimal numbers to words
- Real-time conversion as you type
- Button-triggered conversion
- Clean and responsive Blazor UI

---

## 🧑‍💻 Tech Stack

- Blazor Server (.NET 8+)
- C#
